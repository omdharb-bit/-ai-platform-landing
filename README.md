# 🚀 Helix AI — Next-Gen AI Automation Platform Landing Page

A premium, high-converting SaaS landing page for an AI-driven automation platform built as part of a frontend engineering speed challenge.

🌐 **Live Demo:** https://ai-platform-landing-sable.vercel.app/

---

## ✨ Overview

Helix AI is a modern, responsive landing page showcasing an advanced AI automation platform. The project focuses on frontend architecture, performance optimization, SEO best practices, responsive design, and motion choreography.

The landing page demonstrates how premium SaaS experiences can be built using semantic HTML, isolated state management, and native browser animations while maintaining excellent performance.

---

## 🔥 Features

### 💰 Matrix-Driven Pricing Engine

* Dynamic pricing across:

  * 🇮🇳 INR (₹)
  * 🇺🇸 USD ($)
  * 🇪🇺 EUR (€)
* Monthly and Annual billing modes.
* Automatic annual discount calculations.
* Multi-dimensional pricing configuration matrix.
* Zero hardcoded UI price values.

### 📱 Responsive Bento-to-Accordion Layout

* Desktop: Modern Bento Grid experience.
* Mobile: Fluid Accordion interface.
* Automatic state persistence across layout transitions.
* Maintains active feature context when resizing.

### ⚡ Performance Focused

* Native CSS animations.
* Hardware accelerated transitions.
* Minimal re-render architecture.
* Optimized loading experience.
* Responsive across all major breakpoints.

### 🔍 SEO Optimized

* Semantic HTML structure.
* Open Graph metadata.
* Accessible components.
* Crawlable content.
* Optimized page metadata.

## 🚀 Getting Started

### Clone the repository

```bash

```

### Navigate into the project

```bash
cd -ai-platform-landing
```

### Run locally

Simply open `index.html` in your browser or use a local development server.

Using VS Code Live Server:

```bash
Right Click → Open with Live Server
```
